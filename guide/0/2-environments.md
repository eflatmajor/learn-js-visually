# Chapter #2 - Environments

An environment is basically just a place where you can run JavaScript code. There are lots of options available to us:

## Browser Console (Devtools)

One way to run JavaScript code is by using your browser's built-in development tools, which come with a console where you can type or paste code and see the result.

:::tip Open Devtools
1. open your browser's development tools
    - **Windows/Linux** - <kbd>ctrl</kbd> + <kbd>shift</kbd> + <kbd>i</kbd>
    - **MacOS** - <kbd>cmd</kbd> + <kbd>opt</kbd> + <kbd>i</kbd>
2. look for, and click on, the "Console" tab
3. at the bottom left of the console, you should see a `>`
4. click to the right of it and you'll be able to type or paste in code
5. hit <kbd>enter</kbd> to run the code and see its output
6. your code's output will be displayed after a `<`
:::
